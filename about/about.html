about

my notes is a free mininmal but feature rich text editor/note taking website


built in html css and javascript it is built te be fast, responsive and easy to use


privacy

my notes is built for privacy, you and only you should know what are in those notes. Notes can be stored on-device or on a server.

server

notes are stored on a sql server, notes are secured so only you can read them when you choose to read them.

on-device

notes can also be stored on your device, which means you can access them using another text editor and offline. Storing notes on the server will make it easier to access them on other devices.
